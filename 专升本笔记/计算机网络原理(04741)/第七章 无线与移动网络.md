无线网络的基本结构
	（1）无线主机：运行应用程序的终端设备（计算机、智能手机）。
	（2）无线链路：主机通过无线链路连接到基站或者另一台无线主机。
	（3）基站：关键部分。如蜂窝网的蜂窝塔（cell tower）、IEEE 802.11 的 AP。
	（4）网络基础设施：通常是大规模有线网络，如 internet。
不需要基站，没有固定路由器的移动通信模式是自组织网络
无线链路和无线网络特性
	无线链路有别于有线链路主要表现：
		1）信号强度的衰减：电磁波在穿过物体是强度减弱
		2）干扰：同一个频段发送信号的电波源会相互干扰。
		3）多径传播：接收方收到的信号会变模糊。
		无线链路中的比特差错比有线链路更为常见。采用 CRC 和 ARQ 检错。
移动网络
	移动网络的基本原理
		1、无线网络不一定是移动网络，但移动网络一定是无线网络。
	移动互联网体系结构
		1）归属网络（家网）：一个移动结点的永久居所。
		2）归属代理（家代理）：在一个归属网络中代表移动结点执行移动管理功能的实体。一般配置在路由器上。
		3）外部网络（被访网络）：移动结点所在的非归属网络。
		4）外代理：在外部网络中代表移动结点执行移动管理功能的实体。
寻址
	寻址：移动结点从一个网络移到另一个网络保持地址不变。两种解决方案：
		1）外部网络可以通过向其他所有网络发通告，告诉它们该移动结点正在它的网络中。
		2）将移动性功能从网络核心搬到网络边缘，由该移动结点的归属网络来实现。实际移动网络采用这种方式。
	移动结点每移动到一个新的网络，外部代理向归属代理注册一个新的==COA==
移动结点的路由选择
	数据报如何寻址并转发给移动结点：
		直接路由选择克服了三角路由选择的问题，引入了另两个问题：
			1）需要一个移动用户定位协议，方便通信者代理向归属代理查询活动移动结点的 COA。
			2）移动结点从一个外部网络移动到另一个外部网络时，两种解决方案：创建一个新的协议来通告通信者变化后的 COA；通过锚外部代理来实现。
无线局域网 IEEE 802.11
	IEEE 802.11 系列标准
		IEEE 802.11b
			频率范围:2.4G
			数据率:11Mbit/s
		IEEE 802.11a
			频率范围:5G
			数据率:54Mbit/s
		IEEE 802.11g
			频率范围:2.4G
			数据率:54Mbit/s
		IEEE 802.11n
			频率范围:2.4G&5G
			数据率:600Mbit/s
		4 个无线局域网 IEEE 802.11 标准共同特征
			（1）都使用相同的介质访问控制协议 CSMA/CA。
			（2）链路层帧使用相同的帧格式。
			（3）都具有降低传输速率以传输更远距离的能力。
			（4）都支持“基础设施模式”和“自组织模式”两种。
IEEE 802.11 体系结构
	（1）基站：接入点（AP）。
	（2）基本服务集（BSS）：包含一个或多个无线站点和一个接入点的中央基站。
	无线主机如何发现 AP？
		（1）被动扫描：AP 发送信标帧（AP 的 SSID、MAC 地址）
		（2）主动扫描
IEEE 802.11 的 MAC 协议
	IEEE 802.11 多个站点共享信道的多路访问控制协议
CSMA/CA协议
	DIFS:分布式帧间间隔
	RTS:很短的请求发送控制帧(源目地址,通信所需时间)
	SIFS:短帧间间隔
	CTS:允许发送控制帧,包括本次通信所需时间
	NAV:通信时间内,其他站点不会发送数据,该时间段称为网络分配向量
IEEE 802.11 帧的三种类型
	（1）控制帧
	（2）数据帧
		MAC 首部：30 字节
		帧主体：数据部分、不超过 2312 字节（一般小于 1500 字节）
		尾部：检验序列 FCS、4 字节
		序号：使接收方区分新传输的帧和以前帧的重传。
		持续期：预约信道的一段时间（数据帧和确认帧的传输时间）
		帧控制字段：类型和子类型字段用来区分关联帧、RTS 帧、CTS 帧、ACK 帧和数据帧等
		地址字段:
			前 3 种用在有基础设施的网络中；第 4 种用在自组织网络中。
			当把数据帧发给 AP 时，帧控制字段中的“去往 AP=1，来自 AP=0”。
			当 AP 把数据帧转发出去时，帧控制字段中的“去往 AP=0，来自 AP=1”。
	（3）管理帧

蜂窝网络
	2G 蜂窝网络采用 GSM 系统（全球移动通信系统），增加了 GPRS（通用分组无线服务技术）和 EDGE 技术（增强型数据速率 GSM 演进技术）。
	GSM 标准的空中接口使用 ==FDMA/TDMA== 组合技术
	一个==基站控制器（BSC）==通常服务于几十个收发基站。
	基站控制器及其控制的收发基站构成了 GSM 基站系统（BSS）。
	单个移动交换中心（MSC）包含多大 5 个 BSC。
	一个蜂窝服务提供商的网络由若干个 MSC 构成，并通过称为网关的 MSC 与更大的公共电话网络相连。
蜂窝网络中的移动性管理
	GSM 采用==间接路由选择方法==管理移动性。
		移动用户向某个蜂窝网提供商订购了服务，该蜂窝网络称为用户的归属网络。
		移动用户当前所在网络称为被访网络。
	注册器
		归属网络：归属位置注册器（HLR：home location register）的数据库。包括永久蜂窝电话号码、用户个人信息、用户当前位置信息。
		被访网络：访问者位置注册器（VLR：Visitor location register）的数据库。当前在其网络中移动用户包含一个表项。
		一个呼叫如何定位到被访网络中的一个移动用户的过程：
			（1）通信者拨打移动用户的电话号码。号码前几位可定位到用户归属网络的归属 MSC。
			（2）归属 MSC 收到呼叫，查询 HLR 来确定移动用户的位置。HLR 直接返回漫游号码；归属 MSC 查询 VLR 获取移动站点的漫游号码。
			（3）通过漫游号码，从归属 MSC 到被访 MSC，再从被访 MSC 到为移动用户。
				通信者到归属 MSC，从归属 MSC 到被访 MSC，再从被访 MSC 到为移动用户提供服务的基站，最后到移动用户，呼叫连接建立完成。

移动IP网络
	移动 IP 标准由三部分组成:
		代理发现
			代理发现：当某移动 IP 站点到达一个新网络时（外部网络或者回到归属网络），必须知道外部代理或者归属代理的身份。
			代理发现的两种方式：
				代理通告
					外部代理或者归属代理使用一种现有路由器发现协议的扩展协议来通告其服务。周期性广播一个类型字段为9（路由器发现）的 ICMP 报文。
						归属代理位（H）：该代理是它所在网络的一个归属代理。
						外部代理位（F）：该代理是它所在网络的一个外部代理。
						注册要求位（R）：某个移动用户必须向某个外部代理注册。
						M、G 封装位：是否用除了“IP 中的 IP”的封装形式。
						==转交地址（COA）==字段：转交地址列表。
				代理请求。
					一个移动结点可以通过广播一个代理请求报文来知道代理。该报文是类型值为 10 的 ICMP 报文。
		向归属代理注册
			移动 IP 定义了移动结点或外部代理向一个移动结点的归属代理注册或注销 COA 的协议。
			通过外部代理向归属代理完成注册的 4 个步骤
				（1）当收到一个外部代理通告后，移动结点立即向外部代理发送一个移动 IP 注册报文。一个 UDP 数据报，通过端口 434 发送。
				（2）外部代理收到注册报文并记录移动结点的永久 IP 地址。向归属代理的端口 434 发送一个移动 IP 注册请求报文（UDP）。
				（3）归属代理接收注册请求并检测真实性和正确性。将永久 IP 与 COA 绑定。发送一个移动 IP 注册应答。
				（4）外部代理接收注册应答，将其转交给移动结点。
			移动 IP 定义了移动结点或外部代理向一个移动结点的归属代理注册或注销 COA 的协议

其他典型无线网络简介
	WiMax:全球微波互联接入
		目的：在更大范围内为用户提供媲美有线网络的无线通信解决方案。
		优点：更远的传输距离（可达 50km），更高速的带宽接入（可达 300Mbit/s）。
		劣势：不支持用户在==移动过程中无缝切换==。==服务质量差==、==组网性能低==、产业基础薄弱、无法与传统的蜂窝网络==兼容==
	蓝牙
		IEEE802.15.1 网络，个人区域网络。低功率、小范围、低速率的技术。
			用于互联笔记本、串行设备、蜂窝电话和智能手机。
			IEEE802.15.1 网络以 ==TDM== 方式工作于无许可的 ==2.4GHz== 无线电波段。能够提供 ==4Mbit/s== 的数据速率。
			IEEE802.15.1 网络是自组织网络，不需要网络基础设施来互联 IEEE802.15.1 设备。
		ZigBee
			IEEE802.15.4 是 IEEE 的第二个==个人区域网标准==，称为 ZigBee。
			ZigBee 目标：低功率、低数据速率、低工作周期。
			ZigBee 网络结点分为两类：“简化功能设备”、“全功能设备”。多个“简化功能设备”在一个“全功能设备”控制下。
				（1）“全功能设备”使用==信标帧==以时隙方式控制==多个简化功能设备==”。
				（2）由控制器分配给特定的设备。
